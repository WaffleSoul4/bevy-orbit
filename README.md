# Bevy Orbit
A proof of concept simulation/game with gravity

Very much a work in progress

Uses [Bevy](https://bevyengine.org/) as an engine and [Avian2d](https://docs.rs/avian2d/latest/avian2d/index.html) for physics and collision logic

Have Fun <3

## Usage
Run the following commands to clone and run the project:
  
```sh
git clone https://github.com/WaffleSoul4/bevy-orbit.git 
cd bevy-orbit
cargo run
```
